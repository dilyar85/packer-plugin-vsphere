# VMware vSphere Supervisor Builder

Type: `vsphere-supervisor`
Artifact BuilderId: `jetbrains.vsphere`

This builder deploys VMs with the specified content library image to a vSphere Supervisor cluster.

- It uses the current or provided KUBECONFIG file connecting to the vSphere Supervisor cluster.
- It uses the VM-Service API to deploy the source VM.
- It uses the Packer supported provisioners to configure the VM after deployment.
- A new release will be coming up soon to publish the customized VM to content library.

## Examples

See provided template files in the [examples/supervisor folder](https://github.com/hashicorp/packer-plugin-vsphere/tree/main/builder/vsphere/examples/supervisor).
